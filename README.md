# homebrew-grove

Homebrew tap for [Grove](https://github.com/henrypl/grove) - mobile terminal server for Claude Code.

## Installation

```bash
brew tap henrypl/grove
brew install grove
```

## Usage

```bash
grove              # Start server interactively
grove -b           # Start in background
grove --stop       # Stop background server
grove --help       # Show help
```

## Updating

```bash
brew update
brew upgrade grove
```
